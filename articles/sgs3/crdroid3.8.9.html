Downloads:

First, download the ROM.
crDroidAndroid-7.1.2-20200102-d2att-v3.8.9.zip

How to install this ROM using TWRP:

Boot to TWRP Recovery [VOL UP] [HOME] and [POWER]
Press [Backup]
Slide To Backup >>>
Go [Home]
Press Wipe
Press [Advanced Wipe]
Select [SYSTEM][DATA][CACHE][DALVIK CACHE]
Slide to Wipe >>>
Go [Home]
Press [Install]
Select [crDroidAndroid-7.1.2-20200102-d2att-v3.8.9.zip]
Slide To Install >>>
Press [Reboot System]
ENJOY! (First boot might take from 1 to 5 minutes so be patient)

Open Gapps: [ARM]-[7.1] (Optionnal)

NOTE: Boot the system once before installing GApps. After booting up, reboot to your custom recovery and install GApps
Not doing so might result in a bootloop and you will have to flash again.
